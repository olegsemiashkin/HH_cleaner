# HH Cleaner

Расширение для Chrome, которое автоматически очищает список чатов на HeadHunter.  
Скрипт по очереди открывает верхний чат, выходит из него и повторяет цикл, пока чаты не закончатся.

## Возможности

- Работает на странице чатов HH: `https://spb.hh.ru/chat`
- Поочередно:
  - кликает на первый чат в списке;
  - открывает меню чата;
  - выбирает пункт «Покинуть чат»;
  - ждёт, пока чат исчезнет из списка, и переходит к следующему.
- Запускается вручную по клику на иконку расширения.

## Установка

1. Склонируйте репозиторий или скачайте архив с кодом.
2. Откройте в браузере страницу `chrome://extensions/`.
3. Включите режим разработчика (Developer mode) в правом верхнем углу.
4. Нажмите «Load unpacked» / «Загрузить распакованное расширение».
5. Выберите папку с файлами:
   - `manifest.json`
   - `background.js`
   - `content.js`
   - `popup.html`
   - `popup.js`

## Использование

1. Откройте страницу чатов HH: `https://spb.hh.ru/chat` (обязательное условие).
2. Убедитесь, что вы залогинены в аккаунт.
3. Нажмите на иконку расширения Chrome.
4. В появившемся popup нажмите кнопку запуска (если предусмотрена) либо просто кликните по иконке, если расширение настроено на запуск по клику.
5. Наблюдайте, как чаты последовательно закрываются, пока список не опустеет.

## Структура проекта

- `manifest.json` — описание расширения и разрешений (Manifest V3).
- `background.js` — обработка клика по иконке и запуск `content.js` на активной вкладке.
- `content.js` — логика прохода по списку чатов и выхода из каждого.
- `popup.html` — интерфейс всплывающего окна расширения.
- `popup.js` — связь UI popup с запуском основного скрипта.

## Ограничения

- Работает только на десктопной версии HH на странице `https://spb.hh.ru/chat`.
- При изменении верстки или data-атрибутов на HH могут потребоваться правки селекторов в `content.js`.
