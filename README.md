# HH Cleaner

Расширение для Chrome, которое автоматически очищает список чатов на HeadHunter.  
Скрипт по очереди открывает верхний чат, выходит из него и повторяет цикл, пока чаты не закончатся.

## Возможности

- Работает на странице чатов HH: `https://spb.hh.ru/chat`.
- Поочередно:
  - кликает на первый чат в списке;
  - открывает меню чата;
  - выбирает пункт «Покинуть чат»;
  - ждёт, пока чат исчезнет из списка, и переходит к следующему.
- Запускается вручную по клику на иконку расширения.

## Установка (без архива)

1. Создайте на компьютере пустую папку, например `hh-chat-cleaner`.
2. С этого репозитория по очереди скачайте файлы:
   - `manifest.json`
   - `background.js`
   - `content.js`
   - `popup.html`
   - `popup.js`
3. Поместите все скачанные файлы в созданную папку `hh-chat-cleaner` (файлы должны лежать прямо внутри папки, без вложенных директорий).
4. Откройте в Chrome страницу `chrome://extensions/`.
5. Включите режим разработчика (Developer mode) в правом верхнем углу.
6. Нажмите кнопку «Load unpacked» / «Загрузить распакованное расширение».
7. Выберите папку `hh-chat-cleaner`.

После этого расширение появится в списке иконок Chrome.

## Использование

1. Обязательно перейдите на страницу чатов HH: `https://spb.hh.ru/chat`.
2. Убедитесь, что вы залогинены в аккаунт HH.
3. Нажмите на иконку расширения Chrome.
4. В popup нажмите кнопку запуска (если включена) или просто кликните по иконке, если расширение настроено на запуск по клику.
5. Наблюдайте, как чаты последовательно закрываются, пока список не опустеет.

## Структура проекта

- `manifest.json` — описание расширения и разрешений (Manifest V3).
- `background.js` — обработка клика по иконке и запуск `content.js` на активной вкладке.
- `content.js` — логика прохода по списку чатов и выхода из каждого.
- `popup.html` — интерфейс всплывающего окна расширения.
- `popup.js` — связь UI popup с запуском основного скрипта.

## Ограничения

- Работает только на десктопной версии HH на странице `https://spb.hh.ru/chat`.
